# Introducing NW.js

## Installation

```bash 
npm install -g nw
```

Excuting!! Inside the hello-world-nwjs folder ,run th following command on your terminal:

```bash 
nw
```

## Building Your App for Multiple OSs From a single codebase

```bash 
npm  install -g  nw-builder
nwbuild app/
```
